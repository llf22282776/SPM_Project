1. 项目简介
本项目是北京邮电大学软件学院spm的课程项目。主要是对spm v2.0项目中的缺陷进行修改和添加。

2. 项目功能
本项目添加的功能点如下：
①注册
账号的工号判断是否仅为数字，且名称可以不唯一；
判断密码是否仅包含数字和字母，并且不含特殊字符；
判断邮箱是否合法，且绑定的邮箱唯一；

②登录
仅允许一个终端登录一个角色，不允许登录多个角色；
登录后能够正确显示与该角色权限相符的界面和功能；

③留言版
判断留言内容是否非空；

⑤
